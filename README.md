# Week 2 Pyhton Workshop - Control Flow & Data Types

## Overview
This repository contains my Week 2 Python workshop exercises from the ITEC102 course. The notebook builds on the foundations from Week 1 and introduces core programming concepts related to data types, operators, conditional logic, and loops.

The goal of this workshop was to develop a stronger understanding of how programs make decisions and repeat actions using control flow structures, while also exploring how Python handles different data types and expressions.

---

## Concepts Demonstrated
This notebook includes examples of:
- Python data types (`int`, `float`, `str`, `bool`, `bytes`, `datetime`)
- Type conversion and implicit type casting
- Arithmetic operators
- Comparison and logical operators
- Conditional statements (`if`, `elif`, `else`)
- For loops and while loops
- The `range()` function
- String slicing and formatting
- Program flow control using `break` and `continue`

Exercises in this workshop include:
- Printing greeting messages using formatted strings
- Performing arithmetic calculations and operator demonstrations
- Evaluating comparison and logical expressions
- Creating decision-based programs using conditional logic
- Iterating through lists and ranges using loops
- A temperature checker that categorises weather conditions
- A small Weather & Event Planner mini project

These exercises help demonstrate how basic logic structures can be used to control program behaviour and interact with user input.

## Running the Notebook

## Option 1 - Google Colab (Recommended)

1. Open the notebook in Google Colab
2. Run each cell in order

## Option 2 - Jupyter Notebook

1. Install Jupyter Notebook if not already installed:
```
pip install notebook
```
2. Launch Jupyter Notebook:
```
jupyter notebook
```
3. Open the notebook and run each cell in order.
